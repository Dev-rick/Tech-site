Bootstrap is a CSS framework, which means that it has a lot of pre-written CSS code that you can use in your HTML project. Bootstrap was created by two software developers at Twitter and it soon became the most popular CSS framework. Websites that use Bootstrap look beautiful and modern and are able to adapt to different screen sizes (even to small mobile screens). This ability is called responsive design.
